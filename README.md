## mNTCapture
A mobile app to capture the mobile network packets and display it in real time, it uses the VPN to route the data through the VPN and sniff the packets.

##### - To directly run the app in your device, download the apk from the github folder above and install in you device.


#### Note:- The app is in beta (experimental) phase. Future updates will fix existing bugs and address the compatibility issues, if any.
